Technology Used

NEW CONCEPT: Single page application (SPA) but with the development style of traditional multiple-page using hyperlinks. (Click here to see all pages's hyperlink used in this application)

  • Modular distributed - scalable, flexible, reusable and easy replaceable
  • Microservices - service oriented architecture (SOA)
  • Single page application (SPA) - fast response, better user experience
  • Static web hosting from anywhere - fast, secure and low cost or free (using GitHub+CloudFlare free services)
  • No server side coding - simple development and easy maintenance
  • Data access with APIs - read/write data from/to a cloud database with APIs
  • File access with APIs - download/upload files from/to Amazon S3 storage with APIs
